# Be Global — Digital Marketing Agency Website

A clean, modern multi-page website built for **Be Global**, a digital marketing agency. This project was built as part of a Website Development internship, with a focus on clean design, brand consistency, and a smooth browsing experience across devices.

The site is fully static — no backend or database required — and can be viewed by simply opening the HTML files in a browser, or by hosting it on any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

---

## 🌐 Live Pages

The website consists of 5 pages, all linked together through a shared navigation bar and footer:

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with hero section, services preview, and why-choose-us section |
| About Us | `about.html` | Company story, mission & vision, core values, and team |
| Services | `services.html` | Detailed breakdown of the agency's core services and working process |
| Case Study | `case-study.html` | A sample client success story with results and testimonial |
| Contact Us | `contact.html` | Contact form and business details *(coming soon)* |

---

## 🎨 Design & Branding

The entire site follows a consistent brand identity:

- **Primary (Blue):** `#0168F9` — used for buttons, links, and highlights
- **Secondary (Navy):** `#1A1A2E` — used for headings and primary text
- **Tertiary (Purple):** `#B687F9` — used for accents and gradients
- **Font:** Poppins — a clean, rounded, modern sans-serif

The layout leans on plenty of white space, soft rounded corners, and subtle hover animations to keep things feeling modern and uncluttered rather than busy or corporate.

---

## 🛠️ Tech Stack

This project was intentionally kept lightweight — no frameworks, no build tools, no dependencies. Just clean, hand-written code:

- **HTML5** for structure
- **CSS3** (custom, no framework) for styling — using CSS variables for easy theming
- **Vanilla JavaScript** for small interactive touches (mobile menu toggle, scroll animations)

Everything was written and tested in **VS Code**.
